# 📊 Financials Dashboard | Power BI

Projeto desenvolvido como parte de um desafio prático da DIO (Digital Innovation One), utilizando o dataset Financials.

## 🚀 Objetivo

Criar um relatório analítico no Power BI com visualizações interativas, aplicando conceitos de análise de dados e construção de dashboards.

## 📌 Principais recursos

- 📊 Análise de vendas e lucro
- 📈 Gráfico de tendência mensal
- 🌎 Análise por país, segmento e produto
- 🎯 Uso de indicadores (Bookmarks) para alternar visuais
- 🧭 Navegação entre páginas com botões
- 🎛️ Segmentadores interativos (filtros)
- 🎨 Layout organizado

## 📄 Páginas do relatório

**Página 1 – Sales Report**
- KPIs principais (Total de Vendas, Unidades, Descontos, COGS)
- Tendência de vendas por mês
- Análise por segmento, produto e país

**Página 2 – Profit Report**
- Análise detalhada de lucro
- Distribuição por produto e segmento
- Análise por trimestre

## 🛠️ Ferramentas utilizadas

- Power BI Desktop
- Dataset Financials (Microsoft Sample)
